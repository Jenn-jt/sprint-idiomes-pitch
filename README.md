# Sprint Idiomes — Proposta web

> Prototip confidencial preparat per **She's Digital** com a proposta de redisseny
> per a Sprint Idiomes (Sant Cugat del Vallès).

Aquest repositori conté un prototip HTML estàtic d'una nova web per a l'escola
d'anglès Sprint Idiomes. Es tracta d'una proposta comercial; el client encara
no ha contractat la feina.

## Demo

Per veure el prototip en viu, consulta l'URL de Vercel associada a aquest repositori.

## Estructura

```
.
├── index.html          # Pàgina única amb tot el codi (HTML + CSS inline)
├── LICENSE             # Tots els drets reservats — She's Digital
└── README.md           # Aquest fitxer
```

## Stack

- HTML5 + CSS3 (sense build step, sense dependències)
- Tipografies: Bricolage Grotesque, Nunito, Caveat (Google Fonts)
- Animacions CSS pures (sense JS)
- Responsive fins a 375 px

## Tecnologia futura

Si el client contracta, la web es portarà a **WordPress** amb tema fet a mida
per She's Digital, mantenint la mateixa estètica i estructura.
